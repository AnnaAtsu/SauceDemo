# 🧪 Чек-лист автотестов для SauceDemo

## 🔐 Страница логина (`/`)

| ID | Проверка | Ожидаемый результат |
|----|----------|---------------------|
| **LOG-01** | Вход с валидными учетными данными (`standard_user`/`secret_sauce`) | Перенаправление на страницу `/inventory.html`, отображается список товаров |
| **LOG-02** | Вход с невалидными учетными данными | Остаемся на странице логина, отображается сообщение об ошибке: `"Epic sadface: Username and password do not match any user in this service"` |
| **LOG-03** | Вход с пустым полем username | Сообщение об ошибке: `"Epic sadface: Username is required"` |
| **LOG-04** | Вход с пустым полем password | Сообщение об ошибке: `"Epic sadface: Password is required"` |

## 📦 Страница товаров (Products) — `/inventory.html`

| ID | Проверка | Ожидаемый результат |
|----|----------|---------------------|
| **PROD-01** | Отображение всех товаров | На странице присутствует 6 карточек товаров (`inventory_item`) |
| **PROD-02** | Добавление товара в корзину через кнопку "Add to cart" | Кнопка меняется на "Remove" |
| **PROD-03** | Удаление товара из корзины через кнопку "Remove" | Кнопка меняется на "Add to cart" |

## 🛒 Страница корзины (Cart) — `/cart.html`

| ID | Проверка | Ожидаемый результат |
|----|----------|---------------------|
| **CART-01** | Отображение добавленных товаров | В корзине присутствуют именно те товары, которые были добавлены на странице Products |
| **CART-02** | Удаление товара из корзины через кнопку "Remove" | Товар исчезает из списка корзины |
| **CART-03** | Переход обратно к покупкам по кнопке "Continue Shopping" | Возврат на страницу `/inventory.html` |
| **CART-04** | Переход к оформлению заказа по кнопке "Checkout" | Открывается страница `/checkout-step-one.html` |
| **CART-05** | Пустая корзина | Отображается сообщение, что корзина пуста, или отсутствуют элементы `cart_item` |
| **CART-06** | Количество товаров в корзине соответствует добавленному | Счетчик на иконке корзины равен числу товаров в списке |

## 🏷️ Карточка товара (Product Details) — `/inventory-item.html?id=...`

| ID | Проверка | Ожидаемый результат |
|----|----------|---------------------|
| **DET-01** | Отображение полной информации о товаре | Присутствуют: изображение, название, описание, цена |
| **DET-02** | Кнопка "Add to cart" добавляет товар | После нажатия кнопка меняется на "Remove", обновляется счетчик в корзине |
| **DET-03** | Кнопка "Back to products" возвращает в каталог | Переход на страницу `/inventory.html` |
| **DET-04** | Кнопка "Remove" удаляет товар из корзины | Счетчик корзины уменьшается или исчезает (если был 1 товар) |
